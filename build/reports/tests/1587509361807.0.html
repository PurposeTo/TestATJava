      Описание

      Банк «Открытие» — один из крупнейших банков России, входил в список системно-значимых кредитных организаций, утвержденный Банком России. Банк занимал седьмое место по размеру активов среди всех российских банковских групп. Работал на финансовом рынке с 1992 по 2014 год. Википедия
      Дата основания: 1992 г.
      Дочерние компании: Obshestvo s Ogranichennoj Otvetstvennostyu "Tamozhennaya karta", ЕЩЁ
